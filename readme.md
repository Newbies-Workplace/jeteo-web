# jeteo-web
## Development

### Instalation

* Clone this repo
* Install nodejs
  * For linux users
  ```bash
  nvm use
  ```
  * For windows folks<br/>
    * `install newest nodejs from official website` (and u ready) 
    <br/>or
    *  `nvm-windows` and then
    ```batch
    nvm use [paste here version from .nvmrc] 
    ```
* `npm i`

### Dev server
```bash
npm run dev
```
